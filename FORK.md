# Fork Note

This repository is a fork of [CapSoftware/Cap](https://github.com/CapSoftware/Cap).

This fork is maintained as a Windows desktop workspace with a narrower scope:

- Tauri desktop application
- SolidStart desktop frontend
- Rust core, capture, media, render, and development crates needed by the desktop runtime

Everything outside that scope should be treated as cleanup debt rather than product surface.
